【Meta Quest】国内激活教程

发表于 2024-08-20

关于Quest账号

注意：除了下载安装游戏外，其他所有步骤都需要魔法。

注册方式-邮箱注册

Meta网页版（推荐）
Meta苹果版应用
Meta安卓版应用
*苹果版需要外区商店。
*安卓版如果没有谷歌商店，可使用“出境易”

开启Meta账号两步验证-必须

下载两步验证器（2fas-authenticator）

苹果版
安卓版

开启两步验证

大致步骤：
1、前往Meta个人中心（点击直达）
2、选择“Password and security -> Two-factor authentication -> 你的账号 -> Authentication app”，选择后它会给你一个二维码，不要关闭窗口

3、打开刚才下载的手机软件（两步验证器），点击“+”号，扫描二维码。
当有6位数在你手机软件上生成后回到网页，填写备注保存提交。
4、此时“Authentication app”这里应该显示为“Recommended”，大功告成（暂时）。

激活Meta Quest

必须使用手机软件激活，并给予必要权限（蓝牙，定位，发现本地/附近设备等权限）

下载Meta Horizon应用

Meta苹果版应用
Meta安卓版应用
*苹果版需要外区商店。
*安卓版如没有谷歌商店，可使用“出境易”

开启Meta Quest专用热点

1、需要有网卡并搭载Windows系统的电脑（人话：可以连Wifi的Windows电脑）。
2、下载oculus助手。需要暂时关闭杀软，部分杀软会误报毒，实测安装后没有病毒。
安装完后根据软件提示完成Quest热点开启，并在进入到Quest系统之前不要关闭软件。

3、首次激活建议充值流量包，最低档就够了，实测免费流量包完全不够用。
4、在Quest上连接“oculus助手”上显示的热点，开始自动激活和更新。
5、更新期间它会让你摘下头显在手机上进行其他操作。
建议不要听它的，因为它还会重启一遍，导致白白设置，建议等它全部更新完毕再设置其他的。

手机上继续进行操作

注意：需要魔法
1、Quest更新完后打开Meta手机应用。如果正确给予了应用权限，稍等一会就会自动出现配对的提示框。
如果没有出现配对框，尝试重启应用。
2、激活步骤中关于“分享数据”的选项，选择不分享。其他的按照自己的喜好选择。
3、完成所有选项后，头显中的激活框便会自动消失，正式进入Quest系统。
4、第一时间在Quest设置中关闭系统自动更新。
5、至此，Quest激活和更新已完成，可以等待系统里的原厂App更新完毕后再关闭Quest热点。

成为开发者畅玩游戏

注意：一定要首先开启上方提的的“两步验证器”，否则会永久失去账号。
视频教程：绑定手机号
1、前往Meta网页版个人中心（直达）。
选择“Password and security -> Two-factor authentication -> 你的账号 -> Text message”。
2、先尝试绑定你自己的手机号，注册区号是否选择正确“China(+86)”。
3、如果收不到短信，说明Meta验证服务器又炸了，需要用到虚拟号。
4、我用的是5sim，注册账号后充值1美元（支持国内支付方式）。
充完后选择Facebook服务，再选择香港，等他下发虚拟号。
5、下发后回到Meta个人中心并保持刚才的选项，区号应选择+852，填入虚拟号852后面的数字。
然后发送验证码，等待虚拟号返回验证码并提交。
6、绑定成功后，后续登录选择“两步验证器”进行验证，因为你无法再用这个虚拟号接收后续短信。
7、前往开发者中心，随便填写内容并提交，即可成为开发者。
8、打开手机App，选择“个人头像 -> 右上角打开菜单 -> 设备 -> 选择设备 -> 头戴设备管理 -> 开发者模式 -> 开启开发者模式”。
9、完成。

下载安装游戏

1、需要Windows系统。
2、下载“Quest助手”
3、下载完后使用数据线连接头显和电脑，并在头显内完成权限授权（建议勾上“一律允许”）

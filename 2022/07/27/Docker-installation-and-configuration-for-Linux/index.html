Linux之Docker安装与配置

发表于 2022-07-27 分类于 Other

第一步：检查系统是否已安装Docker，回车后未返回任何信息就可以进行手动安装。

第二步：使用yum安装docker。

第三步：检查安装项无误后，输入y开始安装。

第四步：安装完成（出现“Complete！”）后，启动Docker。

第五步：检查Docker运行状态，出现绿色active字样即正常运行。

第六步：将Docker设置为开机自启。

至此已配置完毕

关闭Docker开机自启指令。
